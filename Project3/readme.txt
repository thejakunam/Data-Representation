README
======

To run LLOYD :
		python lloyd.py wine.data.txt 3 10 output_lloyd.txt
		Here 3 = no. of clusters
		10 = no. of iterations
		
To run KMEANSPP :
		python kmeanspp.py wine.data.txt 3 10 output_kmeanspp.txt
		Here 3 = no. of clusters
		10 = no. of iterations
		
To run SPECTRAL :
		python spectral.py wine.data.txt 3 1.5 output_spectral.txt
		Here 3 = no. of clusters
		1.5 = sigma value
